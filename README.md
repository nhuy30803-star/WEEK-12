# Iris Flower Classification Project

## Overview
This project performs classification on the classic Iris Flower dataset. It uses several machine learning algorithms to predict the species of an iris flower based on its sepal and petal measurements.

## Dataset
The Iris dataset contains 150 samples of iris flowers from three different species:
- Setosa
- Versicolor
- Virginica

Four features are used for classification:
- Sepal length
- Sepal width
- Petal length
- Petal width

## Algorithms Used
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest

## How to Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the classification script:
   ```bash
   python iris_classification.py
   ```

## Results
The script outputs the accuracy and classification report for each model, identifying the best-performing algorithm for the dataset.
